query_waveform_data(sids, signal_names, process_names)[source]

Queries the waveform array data for a given set of sids, signal_names, and process_names.

Results are stored internal to this object.

Parameters:

  • sids (List[int]) – A list of scan database identifiers to query waveform data

  • signal_names (Optional[List[str]]) – A list of the signal names to include data from (GMES, PMES, etc.). If None, all signals are queried.

  • process_names (Optional[List[str]]) – A list of the process names to include data from (names of array transforms, e.g. raw or power_spectrum). If None, all array types are queried.

Return type:

List[Dict[str, Any]]

Returns:

A list of dictionaries each containing the data for a single array of raw or process data from a waveform.

query_waveform_metadata(sids, signal_names, metric_names)[source]

Queries the waveform scalar metadata for a given set of sids, signal_names, and metric names.

Results are stored internal to this object.

Parameters:

  • sids (List[int]) – A list of scan database identifiers to query waveform data

  • signal_names (List[str]) – A list of the signal names to include data from (GMES, PMES, etc.)

  • metric_names (List[str]) – A list of the scalar metad to include in the output (mean, median, etc.). If None, get all.

Return type:

List[Dict[str, Any]]

Returns:

A list of dictionaries each containing the scalar metadata for a single waveform.
